# Dungeon Escape (Unreal Engine 5 - C++)

## Overview
Dungeon Escape is a C++ based Unreal Engine project focused on
core gameplay systems, interaction logic, and modular architecture.

## Features
- Custom character controller
- Collectible system
- Game mode logic
- Modular component-based design

## Tech Stack
- Unreal Engine 5
- C++
- Git version control

## Notes
Heavy content assets are excluded from this repository
